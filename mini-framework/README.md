
A lightweight JavaScript framework built from scratch featuring a Virtual DOM, State Management, and a Routing system.

- **Virtual DOM**: Efficiently updates only the parts of the DOM that changed.
- **Inverted Control**: The framework manages the lifecycle and rendering.
- **Global State**: Reactive data binding across the entire app.
- **Custom Events**: An abstracted event API for cleaner code.
- **Client-Side Routing**: URL synchronization with the app state.
